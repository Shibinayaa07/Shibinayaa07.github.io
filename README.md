# Shibinayaa S — Portfolio

This repository contains a modern single-page portfolio for Shibinayaa S.

Structure:

- `index.html` — single-page site
- `style.css` — minimal styling with dark/light mode
- `script.js` — theme toggle, modal, smooth scroll
- `assets/` — profile image, resume, and project assets

To preview locally (Windows + XAMPP), put this folder in your web root and open `http://localhost/Shibinayaa07.github.io/index.html` or open `index.html` directly in a browser.

Replace `assets/profile.jpg` and `assets/resume.pdf` with real files.
